# 🌐 Futuristic Personal Portfolio

A high-performance, visually stunning personal portfolio built with a **Cyberpunk/Sci-Fi aesthetic**. This project showcases my skills in **Data Analytics** and **Cybersecurity** using interactive 3D elements.

## 🚀 Live Demo
Check out the live site here: [https://vikasmaurya8307.github.io/Portfolio/](https://vikasmaurya8307.github.io/Portfolio/)

## ✨ Features
- **3D Background:** Interactive particle field and wireframe structures built with **Three.js**.
- **Cyberpunk UI:** Custom cursor, neon glow effects, and angled cut-out buttons.
- **Dynamic Stats:** Animated counters for projects and commits.
- **Fully Responsive:** Optimized for mobile, tablet, and desktop views.
- **Integrated Resume:** One-click access to my professional resume.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Modern Flexbox/Grid)
- **Animation:** JavaScript, Three.js API
- **Fonts:** Syne & Space Mono (via Google Fonts)
- **Icons:** Font Awesome 6.0
